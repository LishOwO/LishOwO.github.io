Accueil

Bienvenue sur ce site web où je vais vous parler des différents navigateurs web et moteurs de recherches existants
Pour continuer, vous pouvez cliquer sur les onglets en haut de la page

Il y a plusieurs onglets : Accueil, Navigateurs web, Moteurs de Recherche et Sitographie


L'onglet Accueil vous donne une vue générale du site
L'onglet Navigateurs web vous donne une liste de navigateurs web, triés par ordre chronologique
L'onglet Moteurs de Recherche vous donne une liste de moteurs de recherche, triés par ordre chronologique
Et l'onglet Sitographie vous donne une liste de sources utilisées pour enrichir ces-dites pages web
